Focus Windows

Extension only intervenes during these time slots

How it works
1
Label — give the window a name, e.g. Work or Deep Focus
2
Time range — set start and end time, e.g. 09:00 → 17:00. The extension only activates inside this range.
3
Days — pick which days this window applies. Unchecked days are always free.
4
Block these sites — enter domains to block during this window, e.g. reddit.com, x.com. Leave empty and nothing gets blocked automatically.
5
You can have multiple windows — e.g. a morning block and an afternoon block with different sites.

Binge Guard

Block a site after spending too much time on it

How it works
1
Website — enter the domain you want to limit, e.g. reddit.com
2
Block after — how much time you're allowed before it locks, e.g. 20 min 0 sec
3
Rolling window — the time period it tracks usage over, e.g. 1 hr means "20 min in any 1-hour stretch"
4
Once the limit is hit, the site is hard blocked until the window resets. The block lifts automatically — no manual override.
Strict Blocking
When enabled, existing rules cannot be changed or deleted while a site is blocked. Changes are queued and applied after the block expires. Adding new rules is always allowed.

Whitelist

These are always allowed, even during focus hours

How it works
1
Domain whitelist — type a domain like notion.so to allow the entire site. It will never be blocked, regardless of focus windows or binge guard.
2
Path whitelist — type a path like reddit.com/r/MachineLearning to allow only that specific section. The rest of reddit can still be blocked.
3
Path whitelist is prefix-matched — whitelisting reddit.com/r/programming also allows all posts inside it like reddit.com/r/programming/comments/...
4
Whitelist entries cannot be added for a site that is currently hard-blocked by binge guard — wait for the block to expire first.

Enter a domain (e.g. notion.so) to allow the whole site, or a path (e.g. reddit.com/r/learnprogramming) to allow only that section — bypasses both focus hours and binge guard.
